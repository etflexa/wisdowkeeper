📚 Wisdowkeeper

Uma ferramenta de gestão de conhecimento desenvolvida para auxiliar estagiários e equipes de TI a registrarem, organizarem e reutilizarem soluções para problemas recorrentes — especialmente em ambientes que utilizam ERP.

🚀 Sobre o projeto

O Wisdowkeeper nasceu a partir de uma necessidade real vivenciada durante estágio na área de TI, onde problemas recorrentes eram resolvidos diversas vezes sem um histórico estruturado.

A proposta da aplicação é centralizar esse conhecimento, permitindo que soluções sejam facilmente encontradas e reutilizadas, reduzindo retrabalho e aumentando a produtividade da equipe.

🎯 Objetivos
📌 Registrar soluções para problemas recorrentes
🔍 Facilitar a busca por conhecimento já existente
♻️ Reduzir retrabalho dentro da equipe
📈 Acelerar o aprendizado de novos estagiários
🧠 Criar uma base de conhecimento organizada
