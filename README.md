📚 Wisdowkeeper

Uma ferramenta de gestão de conhecimento desenvolvida para auxiliar estagiários e equipes de TI a registrarem, organizarem e reutilizarem soluções para problemas recorrentes — especialmente em ambientes que utilizam ERP.

🚀 Sobre o projeto

O Wisdowkeeper nasceu a partir de uma necessidade real vivenciada durante estágio na área de TI, onde problemas recorrentes eram resolvidos diversas vezes sem um histórico estruturado.

A proposta da aplicação é centralizar esse conhecimento, permitindo que soluções sejam facilmente encontradas e reutilizadas, reduzindo retrabalho e aumentando a produtividade da equipe.

🎯 Objetivos
📌 Registrar soluções para problemas recorrentes
🔍 Facilitar a busca por conhecimento já existente
♻️ Reduzir retrabalho dentro da equipe
📈 Acelerar o aprendizado de novos estagiários
🧠 Criar uma base de conhecimento organizada

🛠️ Tecnologias utilizadas
Frontend: React
Backend: Node.js
Banco de dados: MongoDB
Estilização: Tailwind CSS
Outros: Cloudflare

⚙️ Funcionalidades
✅ Cadastro de problemas e soluções
🔎 Busca por palavras-chave
👥 Organização por categorias ou temas
📝 Edição e atualização de registros
🔐 Autenticação de usuários
📊 Centralização do conhecimento da equipe

🧩 Como funciona
O usuário registra um problema enfrentado
Adiciona a solução encontrada
Outros membros da equipe podem buscar por esse conteúdo
O conhecimento fica armazenado e reutilizável

💡 Possíveis melhorias futuras
⭐ Sistema de avaliação das soluções
💬 Comentários em registros
🏷️ Sistema de tags avançado
📊 Dashboard com métricas de uso
🤖 Sugestão automática de soluções

🖥️ Como rodar o projeto
# Clone o repositório
git clone https://github.com/seu-usuario/wisdowkeeper.git

# Acesse a pasta
cd wisdowkeeper

# Instale as dependências
npm install

# Execute o projeto
npm run dev
